Description:
	This project aims for tracking and displacing the squirrels in Central
Park in web applications. We imports and uses '2018 Central Park Squirrel Census'
data as our referrence to build and demostrate this project.

Functions:
	Our project have the following functions:
1)	/map	
	that supports viewing the location of the a few squirrels on a street map

2)	/sightings
	that lists the squirrels in our database with their key information

3)	/sightings/<unique_id>
	that displays more detailed information about the selected squirrel

4)	/admin		user = ToolsG10 	password = 12345	
	the admin page for adding a new squirrel with certain attributes

5)	/sightings/stats
	that displays the key attributes we wish to demonstrate
